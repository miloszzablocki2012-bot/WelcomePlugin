MSurvivalWelcome - plugin Paper Java 25

Jak zbudowac plugin:
1. Zainstaluj Java 25 i Maven.
2. Otworz CMD/Terminal w folderze MSurvivalWelcomePlugin.
3. Wpisz: mvn package
4. Gotowy plik bedzie tutaj: target/MSurvivalWelcome-1.0.jar
5. Wrzuć ten .jar do folderu plugins na IceHost i zrestartuj serwer.
6. Tekst zmieniasz w: plugins/MSurvivalWelcome/config.yml

Komenda w grze: brak - plugin dziala automatycznie po wejściu gracza.
